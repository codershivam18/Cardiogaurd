## 📌 Overview

CardioGuard is a machine learning-based application that predicts the risk of heart disease using user health data.

It helps in **early detection** and supports better health decisions.

---

## 🎯 Features

✔ Heart disease prediction  
✔ User-friendly interface  
✔ Real-time results  
✔ ML model integration  
✔ Simple and fast  

---

## 🛠️ Tech Stack

- Python 🐍  
- Machine Learning  
- Flask / Django  
- HTML, CSS, JavaScript
